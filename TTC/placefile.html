How to use TTC placefiles in Gibson Ridge


  • With GR open, find the "Windows" dropdown menu near the top
  • Select "Show Placefile Manager"
  • With the placefile manager window open, click the "+" (plus sign) to open a "Load Place File" window
  • In the white address bar, paste one of the addresses below

If you want to have the different color options available to you, repeat this process by using the "Load Place File" option, then turn your desire colors on/off using the check mark functions in Placefile Manager window


Marker Color Link
Red
 https://meteor.geol.iastate.edu/ams/TTC/placefiles/TTC_Forecast_Placefile_Red.txt
Yellow
 https://meteor.geol.iastate.edu/ams/TTC/placefiles/TTC_Forecast_Placefile_Yellow.txt
Blue

Pink
